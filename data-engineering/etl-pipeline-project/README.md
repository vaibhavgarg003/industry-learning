# ETL Pipeline Project 🔧

## What it does
Extracts data from a public API, transforms it, and loads it into a local SQLite database.

## Project Structure
- `src/` → pipeline source code
- `data/` → raw and processed data
- `logs/` → pipeline run logs
- `tests/` → unit tests

## Tech Stack
- Python, Pandas, SQLAlchemy, Requests
